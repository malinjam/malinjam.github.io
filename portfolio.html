pillowtalk
We were given the theme "Designing for a World of Introverts" in an interactive media course at KTH and chose to design an interactive pillow system to allow airplane passangers to regulate the amount of interaction they had on the plane. Our group did this by designing both a interactive program in each passangers seat monitor and a pillow-like headphone system for comfort. We also followed the double diamond mthodlogogy throughout the project
  • Conducted semistructured interviews to explore user's needs.
  • Used affinity diagram to organise the interviews findings.
  • Developed ideas using the SCAMPER.
  • Created a physical low-fidelity prototype.

progressive web app
As my bachelor's project I investigated the capabilities of progessive web apps in comparison to native apps for comercial flight-booking systems. I did this work at the Mobile Life where I tried to emulate the functionality of their already existing native app in a progressive web app. My thesis is available in DiVa.
  • Researched the definition and implementation of progressive web apps.
  • Mapped the main functions of the native app's booking flow and designed a new flow for a progressive web app.
  • Implemented a progressive webb app with a service worker, local storage and multiple apis.
  • Evaluated the fucntionality of the web app using Google Lighthouse.
Find out more in the github!
